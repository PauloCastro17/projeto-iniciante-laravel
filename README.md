- **24/03** — Dia que criei o projeto que consiste em um Habit Tracker (rastreador de hábitos). Inicei entendendo como criar meu primeiro projeto e principalmente entendo o conceito da estrutura do Laravel. Entendi como criar controladores por comando e como eles são utilizados de maneira a se comunicar com o gerenciamento de rotas. Criei uma view básica para ver como se comporta a renderização e tammbém criei uma pasta de components para poder criar "layouts" repetitivos(o tal do components). E dei inicio a entender como funciona as migrations do Laravel, uma ferramenta útil e principalmente versátil para criação de tabelas.
